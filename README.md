# No Return

**No Return** é um jogo de aventura em linguagem C executado no terminal. O jogador explora diferentes salas, enfrenta inimigos, utiliza uma loja para adquirir recursos e pode acompanhar o histórico das partidas. O projeto foi desenvolvido com foco na prática de programação estruturada, modularização e manipulação de arquivos.

## Funcionalidades

* Sistema de combate.
* Exploração de salas.
* Loja de itens.
* Histórico de partidas.
* Menu de configurações.
* Suporte aos idiomas português e inglês.

## Tecnologias

* Linguagem C
* Biblioteca padrão da linguagem
* Manipulação de arquivos
* Programação estruturada

## Estrutura do projeto

```text
No_return/
├── main.c
├── frontend.c
├── backend.c
├── backend_ptbr.c
├── database.c
├── Config.txt
└── Historico.txt
```

## Como executar

Compile o projeto utilizando um compilador C compatível (como GCC ou Code::Blocks) e execute o arquivo gerado.

```bash
gcc *.c -o NoReturn
./NoReturn
```
